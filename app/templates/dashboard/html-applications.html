Applications

Layout

Web applications should use the centre column layout (as per this example) unless more space is required (for example: an application that requires large maps).

Navigation

If the application is integrated into a website then site navigation should appear identical to the rest of the website. Similarly, if the application is part of a section of a website, then the section navigation column should appear consistent with other pages in that section.

Standalone applications

If the applicaton stands alone independent of a website the site navigation bar may be removed (as per this example). In this case, the left column may be used for a short summary or overview of the application.

Asides

The asides column can be used for supplementary or related information. For applications with "shopping" functionality, it is common for "shopping cart" to appear as the first aside in the asides column.
